# 💰 개인 가계부 & 자산 관리 앱

Netlify에 배포된 순수 HTML/CSS/JS 기반의 개인 가계부 및 자산 관리 웹 앱입니다.  
별도 서버 없이 브라우저 `localStorage`에 데이터를 저장합니다.

---

## ✨ 주요 기능

### 📋 내역 탭
- **수입 / 지출 / 재테크** 3가지 유형으로 내역 분류
- 재테크 항목은 지출 통계에서 제외되며 잔액 계산에 반영
  - `잔액 = 수입 - 지출 - 재테크`
- 카테고리별 내역 추가 / 수정 / 삭제
- 날짜순 정렬 및 유형별 필터 (전체 / 지출 / 수입 / 재테크)
- 월 네비게이터로 월별 내역 조회

### 📊 차트 탭
- 카테고리별 지출 도넛 차트
- 카테고리별 수입 도넛 차트
- 연도별 **월별 현황** 바 차트 (수입 / 지출 / 재테크)

### 💼 자산 탭
- **보유 주식** 등록 및 평가손익 자동 계산 (한국 기준: 상승 빨강 / 하락 파랑)
- **배당금 내역** 등록 (KRW / USD 지원, 자동 환율 변환)
- **현금 / 예금** 계좌별 관리
- 자산 배분 도넛 차트
- **연간 총 자산 추이** 라인 차트 (연도 네비게이터 지원)
  - 주식 현재가 / 수량 변경 이력 자동 저장
  - 현금 / 예금 잔액 변경 이력 자동 저장
  - 이력 기반으로 월별 실제 자산 변동 반영
- **월별 배당금 추이** 바 차트
- 4개 이상 항목은 3개 고정 표시 후 스크롤

### 🗂️ 카테고리
- 기본 카테고리 제공 (식비, 카페, 교통, 쇼핑 등)
- 재테크 전용 카테고리 (재테크, 투자, 보험)
- 커스텀 카테고리 추가 / 수정 / 삭제
  - 기기 이모지 키보드 또는 탭별 이모지 팔레트에서 선택

### 💹 예산 탭
- 카테고리별 월 예산 설정
- 예산 대비 지출 현황 및 초과 알림

## 📌 데이터 저장 방식

- 모든 데이터는 **브라우저 localStorage**에 저장됩니다
- 기기/브라우저가 다르면 데이터가 공유되지 않습니다
- 브라우저 데이터 초기화 시 데이터가 삭제됩니다
